# Profile-likelihood CIs where computable; Wald-CI fallback only when profiling
# fails. The president fixed-effects spec has complete separation on this small
# sample, so its profile CI cannot be computed - this keeps that spec in the
# robustness table (as pre-registered) instead of aborting the render. The other
# three specs are unaffected and keep their exact profile intervals.
tidy_ci <- function(m) {
  tryCatch(
    tidy(m, conf.int = TRUE),
    error = function(e) {
      z <- qnorm(0.975)
      tidy(m) |> mutate(conf.low  = estimate - z * std.error,
                        conf.high = estimate + z * std.error)
    }
  )
}

The claim being tested

A familiar story about American politics: when a president faces a Congress controlled by the other party, they stop legislating and start decreeing, reaching for the executive order to act without lawmakers. It’s intuitive, it’s repeated constantly, and it has serious scholarship behind it (Howell, Power Without Persuasion; Mayer, With the Stroke of a Pen).

This is a small, deliberately narrow test of one measurable piece of that story: does the raw volume of executive orders rise under divided government?

NoteThis analysis was pre-registered

The hypothesis and the full model specification were frozen before any data was pulled or seen. See PRE_REGISTRATION.md. That means the result below was not reverse-engineered to be interesting. A clean null is reported as honestly as a clean positive.

What counts as “divided,” and where the numbers come from

Every number here traces to a public source:

  • Executive orders: the complete 1953–2025 record, counted by signing date. The Federal Register’s search API only has full coverage from 1994 on, so the pipeline combines its pre-1994 disposition tables with the 1994+ API, then validates per-president totals against published counts and refuses to run if the data is incomplete.
  • Party control of each chamber: hand-verified per Congress (to correctly handle caucusing independents and VP-tiebreak Senates that a naive party count miscodes), and independently cross-checked against voteview.com DW-NOMINATE records.
  • Presidential party: a 14-row administrations table.

A year is coded divided when the president’s party does not control both chambers.

The result

Code 
res <- tidy(nb, conf.int = TRUE) |> filter(term == "divided_government")
irr <- exp(res$estimate); lo <- exp(res$conf.low); hi <- exp(res$conf.high)
sig <- (res$conf.low > 0) | (res$conf.high < 0)   # CI excludes zero?
dir <- if (res$estimate > 0) "more" else "fewer"
verdict <- if (sig && res$estimate > 0) {
  "supports H1: divided government is associated with measurably more executive orders."
} else if (sig && res$estimate < 0) {
  "runs *against* H1: divided government is associated with measurably fewer executive orders."
} else {
  "does **not** support H1: the divided-government association is not statistically distinguishable from zero in this sample."
}

The cleanest specification shows no reliable effect. Comparing each president against himself with president fixed effects, moving between unified and divided control is associated with essentially no change in how many executive orders a president issues (within-president estimate −0.07, 95% CI includes zero). The popular claim that presidents govern by decree when they lose Congress is not supported here. That null is not bulletproof, though: it turns on how a handful of transition years are coded, which the section How fragile is this null? works through in full.

The pooled estimate points the other way, and does not survive that check. In the pre-registered primary model (pooled negative binomial), a divided-government year carries about 0.8× the executive-order rate of a unified-government year (95% CI 0.68–0.94), that is, fewer orders. Taken by itself, the data runs against H1: divided government is associated with measurably fewer executive orders. But it is the pooled estimate only: it does not survive president fixed effects, so it is best read as a between-president composition pattern (some presidents simply governed in high-EO eras under unified control), not as evidence that divided government itself changes how often a president reaches for the pen. The conclusion the analysis lands on is “no reliable effect,” not “fewer.”

In the pooled models (negative binomial, Poisson, OLS), divided-government years carry a lower executive-order rate than unified years, about 0.80× in the primary model, statistically distinguishable from zero. That already cuts against H1, which predicted more unilateral action under divided government. But this null is not robust to coding choices. An independent coding disagrees with ours on three of seventy-three years, and adopting it in full would flip the fixed-effects estimate to a significant negative effect. The honest read, developed in the next section, is that neither the effect nor its absence is robust to how a handful of transition years are coded.

How fragile is this null?

fe_irrp <- function(p){ m <- glm.nb(update(EO_FORMULA, . ~ . + factor(president)), data = p)

setyr <- function(p, yrs, v){ p$divided_government[p$year %in% yrs] <- v; p }
panel_cap <- panel; panel_cap$divided_government <- cap$cap_divided[match(panel$year, cap$year)]
fe_ours <- fe_irrp(panel)
fe_j    <- fe_irrp(setyr(panel, c(2001,2002), 0))   # CAP's Jeffords convention only
fe_2018 <- fe_irrp(setyr(panel, 2018, 1))           # CAP's 2018 only
fe_cap  <- fe_irrp(panel_cap)                        # full CAP coding
irrp <- function(x) sprintf("IRR %.3f, p = %.3f", x["irr"], x["p"])

To see how sturdy the “no reliable effect” conclusion is, I checked it against an independent coding of the same orders: the Comparative Agendas Project executive-orders dataset (1945-2025), which codes each order with its own divided-government flag. Collapsed to the year level, it agrees with our coding on 70 of 73 years. But the three disagreements are enough to move the headline.

Under our coding, the president-fixed-effects estimate shows no reliable effect (IRR 0.936, p = 0.364). Under CAP’s full coding, the same model would show a statistically significant negative effect (IRR 0.840, p = 0.036), which would overturn the headline from “no reliable effect” to “reliably fewer orders under divided government.”

Because the disagreement is only three years, it is worth seeing exactly how each one moves the fixed-effects estimate:

Adopting CAP’s 2001-2002 coding alone leaves the null intact (p = 0.200). Adopting CAP’s 2018 coding alone leaves it intact (p = 0.114). Only both together cross into significance (p = 0.036).

Our reasoning, without hedging:

  • 2001-2002 is a genuine convention difference. CAP codes by which party organized the chamber; we hand-code actual control. After Senator Jeffords left the Republican caucus in mid-2001, the Senate was Democratic-controlled, so we code 2001-2002 as divided. This choice was frozen in advance (pre-registration, limitation 3), not chosen to produce a result.
  • 2018 is not a convention difference. It is an apparent error. 2018 falls in the 115th Congress, which was unified Republican control. CAP files 2018’s orders under the 116th Congress, which did not convene until January 2019. We do not adopt a coding we believe is mistaken in order to obtain significance.

So the null survives every individually defensible recoding, and breaks only under a combination that includes the apparent error. That is the most defensible reading of the evidence. But the plain fact that a three-year recoding in a seventy-three-year series can flip the result means the honest conclusion is the one stated above: neither the effect nor its absence is robust to coding choices. A longer series, or a settled convention for mid-Congress control flips, would be needed to say more.

What this does and doesn’t show

  • Counts ≠ significance. This measures the volume of unilateral action, not its weight. A single consequential order and a routine administrative one count the same. Raw counts are known to undercount the orders that matter most.
  • EOs are one channel. Memoranda and proclamations are other unilateral tools, out of scope here.
  • Small sample. ~73 years means wide confidence intervals; that uncertainty is shown, not smoothed over.
  • Mid-Congress control flips (e.g., the 2001 Senate) are coded at the Congress level and tested in robustness.
  • Completeness was the hard part. An initial pull silently captured only ~1,485 of the ~3,800 orders because of the API’s pre-1994 gap, which by itself faked a result. The analysis proceeds only after a per-president completeness check passes.
